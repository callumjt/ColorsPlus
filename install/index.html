There are currently 2 ways to install Colors+:

Method 1:
step 1: go into the stylesheet you want to use cp in.
step 2: copy "@import "https://raw.githubusercontent.com/callumjt/ColorsPlus/main/colors.css"" into the stylesheet.
step 3: to use the colors you have you put --cp-"name of color".

Method 2:
step 1: go to the Github repo.
step 2: download "colors.css".
step 3: go to the css file you want to use cp in.
step 4: copy "@import 'location of colors.css'" to the top of it.
